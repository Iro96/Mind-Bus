# Mind-Bus Documentation

Welcome to the Mind-Bus docs folder. This documentation is designed to help new contributors understand the project, the code structure, how to get started, and where help is needed.

## Available documents

- `overview.md` — project introduction and goals
- `structure.md` — repository and module structure explained
- `tutorial.md` — setup and usage guide for developers
- `issues.md` — current problems, missing features, and frontend/backend connection notes

## How to use

Read `overview.md` first for high-level context, then `structure.md` for the code architecture. Use `tutorial.md` to run the project locally, and check `issues.md` before contributing.
